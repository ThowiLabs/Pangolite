Crear cliente de sistema

Genera el ID y token que se instalarán en un servidor NAT, VPS del proyecto o red remota. Este cliente de sistema no publica nada por sí solo; solo habilita conexión saliente hacia Pangolite.

Datos del cliente de sistema
Identidad de conexion remota.
Elige el sistema donde instalaras el cliente para mostrar solo los comandos correctos.
Como se usa
  1. Creas el cliente de sistema.
  2. Copias ID y TOKEN en el servidor NAT/remoto.
  3. El agente se conecta hacia Pangolite de forma saliente.
  4. Al crear un recurso, eliges este cliente como origen si el backend vive en ese servidor.
